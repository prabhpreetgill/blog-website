# blog-website

A Blog website that allows users to add blogs is a web application that enables users to share their thoughts and ideas with the world. The website is built using EJS, Node.js, and MongoDB.

EJS (Embedded JavaScript) is a templating engine that allows you to embed JavaScript code in HTML. It is used to generate dynamic HTML pages based on data passed to it from the server. Node.js is a JavaScript runtime that allows you to run JavaScript on the server-side. MongoDB is a NoSQL database that allows for storing and querying unstructured data in the form of JSON documents.

The website features a simple and intuitive interface that allows users to add new blogs by filling out a form. Users can add a title, and a body of the blog post. The website also has a separate page for each individual blog post, where users can see the full post. 

Overall, this Blog website is an excellent platform for users who want to share their thoughts and ideas with the world, it's easy to use, and it's flexible, making it a great option for users who want to start their own blog or for communities that want to share their knowledge and experiences.
